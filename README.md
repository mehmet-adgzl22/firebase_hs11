# firebase_hs11 Proje Özellikleri
<ul>
  <li>Bu projemiz, basit bir üyelik sistemi oluşturan bir React uygulamasıdır. Kullanıcılar kayıt olabilir, giriş yapabilir ve çıkış yapabilirler. </ul>
